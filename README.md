# 🚀 Azure Cloud Project – Windows Server VM Infrastructure Setup

## 📌 Scenario
An organization requires a Windows-based cloud server with a web server installed and validated, secure remote administration, and proper separation of operating system and data storage for future application use.

---

## 💡 Solution
Deployed a **Windows Server Virtual Machine on Microsoft Azure**, configured **IIS Web Server**, and validated the setup using the default IIS page.

Implemented cloud storage best practices by attaching and configuring an **additional data disk** to separate application/data storage from the OS disk, along with **secure networking and RDP-based server management**.

---

## ⚙️ Key Implementations

- Azure Virtual Machine deployment
- Windows Server configuration
- IIS Web Server installation & validation
- OS disk and data disk separation
- Network Security Group (NSG) rules
- RDP-based server administration

---

## 🛠 Technologies Used

- Microsoft Azure
- Windows Server
- IIS Web Server
- Azure Networking (NSG)
- Remote Desktop Protocol (RDP)
